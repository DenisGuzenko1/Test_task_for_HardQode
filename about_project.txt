Это задание написано на Python, с помощью фреймворка Django и Django REST framework. Также использовалась встроенная база данных SQlite.
В файле test_task_for_HardQode\video_lessons\index\urls.py прописаны маршруты для получения API.
В файле requirements.txt прописаны все необходимые для работы связи.
Для установки requirements.txt сначала необходимо создать виртуальное окружение и установить requirements.txt.
Сделать это можно с помощью команды: pip install -r requirements.txt.
После этого можно запустить проект с помощью команды:
python manage.py runserver.
После этого стоит перейти по ссылке, которая появится в терминале.
http://127.0.0.1:8000/
